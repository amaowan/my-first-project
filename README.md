## 小组作业中涉及的数据来源、处理过程、分析内容与可视化设计说明
### 数据来源：
数据爬取自CCTV央视网，主要内容包括具体奥运会，得奖日期，具体项目，奖牌类型，获得者，结合官方公布的个人资料，借助AI，得到运动员，得奖时的年龄，来自省份，（“省份”指运动员代表参赛的注册单位 通常为出生地或长期训练地）。本次作业的原始数据整理为Excel 文件中的夏奥会和冬奥会两个工作表，我们先对原始字段进行了检查，确认可用于分析的核心字段，再按照保留全部记录开展后续处理。由于部分奖牌是团体奖牌，而我们没有进行单人奖牌整理、奖牌去重，实际每届奥运会所获得的奖牌数与我们进行可视化呈现的数量不相符。

### 处理过程和分析内容：
基于公开奖牌明细整理夏奥会和冬奥会数据，我们围绕夏奥会和冬奥会的项目结构、时间趋势、人群特征与区域分布这几个核心分析的出发点，对原始表的数据按不同角度进行整合。夏奥会_项目届次透视用于把夏奥会全部项目按历届进行透视统计，便于图1观察哪些项目在不同奥运周期中持续有奖牌产出。省份汇总用于把夏奥会和冬奥会按赛季、省份、金银铜和总计进行汇总，便于图2观察区域贡献。全部奖牌_年龄明细保留了每一条奖牌记录的年龄、性别、奖牌、赛季和项目信息，便于图5做单点级可视化。项目性别汇总则把每个赛季下每个项目的女性奖牌数、男性奖牌数、总量和平均年龄整理出来，用于图6做气泡散点图。

### 可视化设计说明：
图表设计上，我们尽量避免所有图都使用简单的图的呈现，而是结合数据特点选择不同形式。图1使用三维柱图表达夏奥会项目与届次的对应关系，高度表示奖牌记录数。图2使用矩形树图表达夏奥会和冬奥会在省份维度上的总量结构。图3使用历届趋势图观察两类赛事在不同周期上的奖牌总量变化。图4使用雷达图对比两类赛事的总量、覆盖面和结构特征。图5使用年龄蜂群图表现不同奖牌记录在年龄上的分布。图6使用气泡散点图同时比较女性奖牌数、男性奖牌数和项目总量。
具体情况介绍：
图1：
从项目维度看，奖牌数呈现明显分层：乒乓球、举重柱体高度显著领先，处于第一梯队；拳击、摔跤次之，为第二梯队；足球、网球、现代五项、高尔夫等项目柱体普遍偏低，其中高尔夫最少，属第三梯队。这一差异源于项目设项规则，多小项的乒乓球、举重奖牌产出量远高于小项极少的高尔夫等项目。时间维度上，各项目奖牌数届次间整体稳定，反映出奥运设项的连续性。受3D视角影响，部分柱体存在遮挡，后续可结合二维图表细化分析，为优势项目研究提供参考。
图2：
从1988年到2026年的趋势来看，我国夏奥会的奖牌总数一直远高于冬奥会，整体规模差距非常明显。夏奥会的奖牌数在2008年北京奥运会达到峰值，之后虽有回落，但始终维持在较高水平，说明夏季项目的整体竞争力稳定且强劲。相比之下，冬奥会的奖牌总数长期处于低位，虽然整体呈缓慢上升趋势，但增长幅度有限，波动也更明显，尤其依赖少数几届赛事的集中突破，体现出冬季项目整体发展仍处于相对弱势的阶段，更多依靠重点项目的爆发式表现，尚未形成像夏季项目那样广泛且稳定的竞争优势。
图3：
夏奥会奖牌分布更为广泛，参与省份多，区域覆盖度高。辽宁、江苏、山东、广东、北京等多个省份均贡献了较多奖牌，整体格局较为均衡，体现出夏季项目在全国范围内的普及性与发展基础。
冬奥会奖牌则高度集中，来源省份数量明显更少。吉林、黑龙江、辽宁等东北地区省份占据了奖牌的绝大部分份额，呈现出显著的地域聚集特征，反映出冰雪运动对气候、传统与资源条件的强依赖性。
对比来看，夏季项目已形成多省协同发展的格局，冬季项目仍以传统冰雪强省为核心，区域发展的均衡性有待提升。
图4：
从这张雷达图可以看出，夏奥会在奖牌总量、项目覆盖和省份覆盖这几个核心维度上，都明显领先于冬奥会，说明夏季项目的整体规模和参与面要广泛得多。同时，夏奥会的女性占比和金牌占比也更高，反映出它的优势来源更加多元、数据结构也更丰富；相比之下，冬奥会虽然在团体占比上和夏奥会差距不大，但整体指标都处于较低水平，优势更多集中在少数项目和地区，结构上显得更单一。整体来看，我国夏季项目的发展成熟度和均衡性都明显优于冬季项目。
图5：
全部奖牌获得者年龄集中在20岁到27岁之间。其中夏奥会金牌主要分布在21岁到26岁，冬奥会金牌获得者年龄由于数据量比较少，主要集中在26岁；夏奥会银牌获得者年龄分布在20岁到25岁，冬奥会则集中在22岁；夏奥会铜牌获得者年龄分布在21岁到27岁，冬奥会主要在20岁到22岁。
图6：
从分布看，夏奥会项目气泡更分散，男女奖牌结构差异显著：乒乓球、羽毛球等项目男女奖牌数均较高，结构相对均衡；排球、篮球项目女性奖牌数明显偏高，而体操、举重等项目男性奖牌数更突出。冬奥会项目则集中在少数项目，短道速滑的奖牌总量与男女均衡性表现突出。部分项目气泡仅落在一侧，与团体项目设置、记录口径有关，更适合解读奖牌结构特征，而非严格的人数结论。整体来看，夏奥会项目性别结构差异远大于冬奥会。

## 在整个项目流程中在哪些环节如何使用了什么AI
本次网页可视化作业结合HTML、CSS、JavaScript以及ECharts、D3技术实现。在数据爬取阶段，我们使用通义千问获取运动员得奖时的年龄，省份。其他过程中适当使用Codex作为辅助工具，主要用来快速搭建基础代码框架、生成图表初始代码、梳理简单交互逻辑与代码结构，减少重复基础编码工作，提升效率。
AI仅负责初稿和基础模板的产出，整体项目设计、内容把控、细节优化等由我们小组成员共同完成。
首先，原始Excel数据的理解、字段筛选、数据梳理与预处理完全由自主完成，结合作业需求自行确定分析方向、筛选可用指标、规划不同图表的表达用途，设计数据整理与中间表的处理逻辑。
针对AI生成的初始代码和图表样式，我们进行了大量手动修改与迭代优化。包括手动调整页面布局、配色风格、字体样式，修改图表标题、坐标轴标签、图例展示、文字注释，更换不合适的图表类型，优化数据展示逻辑，修正代码冗余与不合理写法，让可视化效果更贴合题目要求、数据表达更清晰易懂。
同时，页面文案撰写、图表分析解读、整体内容排版、作业说明文档的整理撰写，都由小组独立思考、反复修改完善。
整体来看，AI只作为代码辅助工具使用，只承担机械性、基础性的代码搭建工作。核心的数据分析思路、可视化方案设计、视觉美化、逻辑调整和内容完善，均依靠我们的思考与手动优化完成，保证作业的原创设计与自主完成度。
